부모 자식과 목록

1. ol- li : 그풉핑 - > 번호리스트 용도
2. ul - li : 그풉핑 -> 번호없는 리스트 용도
3. table - tr - td 태그
<\table\> : 격자로 나눠진 표를 만드는 tag <\th> : table head를 의미, table 태그의 자손이다. 열의 항목들이 무엇을 의미하는 지 정의할 때 사용한다
<\tr> : table row 를 의미, table 태그의 자손이다. 화면에서 보일 때 아래로 내려간다.
<\td> : table cell 의미, tr 의 자손이며 열로 나열된다.
<\caption> : table의 제목 , 표이름을 나타낼 때 사용한다(잘 사용하지는 않는다)
a tag html specipication : 공식 설명서 W3C에서 발표하는 html국제 표준안
link tag : 웹의 문서를 연결해주는 태그
